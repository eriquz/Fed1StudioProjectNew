Front End Development

Front Development is the tier of fullstack web development that development that deals with visual element of an application that a user interacts with. The main three used in frontend development are HTML,CSS, and Javascript.these are the only Languages a browser can understand.
IF you to compare this tier of development to a human body, it woild be like the skin.

HTML

Hyper Markup language (HTML) is used to define the structure of the content on a web page. It is arguably the most important fronted develpoment language, because the other two languages use HTML elements to perform thier functions.

CSS

Cascading stylesheets (CSS) is used for formatting and layout of web pages. It was introduced as a way to seperate the concerns of markup (HTML) and formatting (CSS) into separate files.

JS

JavaScript (JS) allows us to make our web page more interactrive. its primarly used to provide the ability to perform programming logic in the frontend, react to events, and validate form data.
